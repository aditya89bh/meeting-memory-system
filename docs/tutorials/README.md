# Tutorials

Step-by-step guides that take you from zero to a deployed, production-ready Meeting
Memory System. Each tutorial is self-contained and uses the bundled
[example data](../../examples/) so you can follow along without writing transcripts.

| # | Tutorial | You will learn to |
|---|---|---|
| 1 | [Getting started](getting-started.md) | Install the package and run the guided demo. |
| 2 | [Import meetings](import-meetings.md) | Turn transcripts into stored memory. |
| 3 | [Searching memory](searching-memory.md) | Run ranked retrieval and inspect why results matched. |
| 4 | [Using the graph](using-the-graph.md) | Explore people, projects, and relationships. |
| 5 | [Generating insights](generating-insights.md) | Produce intelligence reports and recommendations. |
| 6 | [Running automation](running-automation.md) | Schedule recurring import + analysis pipelines. |
| 7 | [REST API](rest-api.md) | Serve and call the HTTP API and dashboard. |
| 8 | [Python SDK](python-sdk.md) | Use the client in local and HTTP modes. |
| 9 | [Docker deployment](docker-deployment.md) | Build and run the container image. |
| 10 | [Production deployment](production-deployment.md) | Operate it reliably with backups and metrics. |

New to the project? Start with [Getting started](getting-started.md) and work down.
